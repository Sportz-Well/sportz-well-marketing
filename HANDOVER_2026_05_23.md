## Critical API Convention — Read Before Touching Any Agent

`ask_with_usage()` in `services/anthropic_client.py` takes:
- `system_prompt` (not `system`)
- `user_prompt` (not `user`)
- Returns a **dict** with keys: `text`, `input_tokens`, `output_tokens`, `web_searches`, `error`
- **Never unpack as a tuple. This bug has been hit before.**

`get_active_product()` returns dict with:
- `product_id` (NOT `id`)
- `product_name` (NOT `name`)
- Every page and agent must use these keys exactly.

## Architecture

One app, one database, one UI. Seven modules coordinate via the database:

```
Researcher   → research_items table
Strategist   → story_angles table
Copywriter   → drafts table
Editor       → editor_reviews table
Media        → media_briefs table
Scheduler    → schedule table
Orchestrator → calls all agents, no own table
```

Agents share: the DB (handoff via rows, not memory) + the Anthropic client wrapper.
