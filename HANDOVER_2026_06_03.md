## Critical API Convention

`ask_with_usage()` takes `system_prompt` and `user_prompt` (not `system`/`user`).
Returns dict with keys: `text`, `input_tokens`, `output_tokens`, `web_searches`, `error`.
**Never unpack as a tuple.**

`get_active_product()` returns dict with `product_id` (NOT `id`) and
`product_name` (NOT `name`).

**PostgreSQL SQL rules (all agents must follow):**
- NEVER use `INSERT OR REPLACE INTO` (SQLite-only) → use plain `INSERT INTO`
- NEVER use `datetime('now')` → use `datetime.utcnow().strftime("%Y-%m-%d %H:%M:%S")`
- NEVER use `sqlite3.OperationalError` → use `Exception`
- `?` placeholders are auto-converted to `%s` by `services/database.py`
